================================================== STUDENT SYSTEM

Description: A Python console program that manages data according to the user's needs.

Requirements:

Python 3.x installed
No external libraries required

How to run it:

Open a terminal in the folder where the app.py file is located and import system.py

Type the following command and press Enter:

python app.py

Follow the menu instructions.

Available options:

1. Add student
2. Show list
3. Search student
4. Update data
5. Delete student
6. Exit

Select the number of the required fields.

The program will display the result.

You can continue requesting data.

To exit, select option 6.

Notes:

Negative values ​​and zero are not accepted.
