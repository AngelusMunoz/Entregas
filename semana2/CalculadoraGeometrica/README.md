================================================
         CALCULADORA GEOMETRICA
================================================

Descripcion:
  Programa de consola en Python que calcula
  propiedades de figuras geometricas 2D y 3D
  segun los datos que el usuario ingrese.

------------------------------------------------
Requisitos:
------------------------------------------------
  - Python 3.x instalado
  - No requiere librerias externas

------------------------------------------------
Como ejecutarlo:
------------------------------------------------
  1. Abre una terminal en la carpeta donde
     este el archivo CalculadoraGeometrica.py

  2. Escribe el siguiente comando y presiona Enter:

       python CalculadoraGeometrica.py

  3. Sigue las instrucciones del menu.

------------------------------------------------
Figuras disponibles:
------------------------------------------------

  Figuras 2D:
    1. Triangulo General
    2. Circulo
    3. Rombo
    4. Rectangulo
    5. Triangulo Rectangulo

  Figuras 3D:
    6. Cubo
    7. Esfera
    8. Piramide cuadrangular regular
    9. Cilindro

------------------------------------------------
Como usarlo:
------------------------------------------------
  1. Selecciona el numero de la figura a trabajar.
  2. Selecciona que deseas calcular.
  3. Ingresa los valores que el programa pida.
  4. El programa mostrara el resultado.
  5. Puedes seguir calculando o regresar al menu.
  6. Para salir selecciona la opcion 10.

------------------------------------------------
Notas:
------------------------------------------------
  - No se aceptan valores negativos ni cero.
  - Los lados del triangulo deben cumplir la
    desigualdad triangular para ser validos.
  - Los angulos del triangulo deben sumar 180.
